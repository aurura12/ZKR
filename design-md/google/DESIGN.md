# EBOM 管理系统 — 前端界面设计文档

> 风格：Google Material Design 3  
> 框架：React 18 + TypeScript + Tailwind CSS + shadcn/ui  
> 图标：Material Symbols  
> 字体：Roboto（英文）、Noto Sans SC（中文）

---

## 一、设计系统概述

### 1.1 设计理念
- **清晰优先**：EBOM 数据量大，界面以信息层级清晰为第一目标
- **效率导向**：减少点击次数，常用操作一步直达
- **专业克制**：避免过度装饰，用颜色和字体权重传达状态

### 1.2 色彩系统（Material Design 3 Token）

```
Primary:        #1A73E8  (Google Blue)
On Primary:     #FFFFFF
Primary Container:   #D3E3FD

Secondary:      #5F6368  (中性灰，用于次要操作)
On Secondary:   #FFFFFF

Surface:        #FFFFFF  (卡片、对话框背景)
Surface Variant:#F1F3F4  (表格交替行、侧边栏)
Background:     #FFFFFF  (页面背景)

Error:          #EA4335  (删除、错误提示)
Success:        #34A853  (成功、通过状态)
Warning:        #FBBC04  (警告、待审批)
Info:           #4285F4  (提示、信息)

Outline:        #DADCE0  (边框、分割线)
Text Primary:   #202124  (主标题、正文)
Text Secondary: #5F6368  (次要信息、时间戳)
Text Disabled:  #9AA0A6  (禁用状态)
```

### 1.3 字体层级

| 层级 | 大小 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| Display Large | 32px | 400 | 40px | 页面大标题 |
| Headline Large | 22px | 500 | 28px | 模块标题 |
| Headline Medium | 18px | 500 | 24px | 卡片标题 |
| Title Large | 16px | 500 | 24px | 列表项标题 |
| Body Large | 14px | 400 | 20px | 正文、表格内容 |
| Body Medium | 13px | 400 | 20px | 次要正文 |
| Label Large | 14px | 500 | 20px | 按钮文字 |
| Label Medium | 12px | 500 | 16px | 标签、徽章 |

### 1.4 间距系统

```

```

### 1.5 阴影层级

```
Level 1: 0 1px 2px 0 rgba(60,64,67,0.1)      — 按钮 hover
Level 2: 0 1px 3px 0 rgba(60,64,67,0.15)     — 卡片

```

## 八、权限设计

```
ADMIN (系统管理员)
  └─ 全部权限

EBOM_MANAGER (EBOM管理员)
  └─ 审批变更申请
  └─ 管理基础数据
  └─ 查看全部数据

PD_LEAD (专业负责人)
  └─ 管理本部门零件
  └─ 提交变更申请
  └─ 查看本部门数据

SQE (SQE工程师)
  └─ 查看供应商信息
  └─ 查看质量相关字段

PROCUREMENT (采购工程师)
  └─ 查看货源信息
  └─ 查看价格相关字段

VIEWER (查看者)
  └─ 只读访问
  └─ 敏感字段脱敏
```

**实现方式：**
- 后端控制数据可见性（column_visibility 表）
- 前端根据用户角色动态渲染菜单和按钮
- 按钮级权限：无权限时按钮禁用或隐藏
